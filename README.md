# Project Noctis

The personal portfolio and showcase for **Night Cats**.

## Tech Stack

- **Vue 3 + TypeScript**
- **Vite**
- **TresJS** (3D)
- **GSAP** (Animations)
- **Lenis** (Smooth Scroll)
- **Tailwind CSS** (Styling)

## Setup

```bash
npm install
npm run dev
```

## Deployment

Automated via GitHub Actions to GitHub Pages.
Push to `main` branch to deploy.
